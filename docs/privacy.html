What to-you does

to-you is a Pomodoro timer that lives in your Mac menu bar. It tracks the time remaining in a focus or break session, plays an ASCII weather scene while you work, and reminds you when it's time to step away.

All of this happens entirely on your Mac. Your preferences — timer duration, weather scene, font — are stored using macOS's standard UserDefaults system, locally on your device, readable only by the app, and never transmitted anywhere.

Data we collect

None. to-you does not collect any data from any user at any time.

Network access

to-you makes no network requests. It has no server, no backend, and no cloud component. The app does not phone home, check for updates over the network, or transmit any information to any destination.

Third-party services

to-you contains no third-party analytics libraries, advertising SDKs, crash-reporting frameworks, or any other external code. The only frameworks used are Apple's own system libraries — SwiftUI, AppKit, and Combine.

Children's privacy

Because to-you collects no data of any kind, it presents no privacy risk to users of any age, including children.

Changes to this policy

If to-you ever changes its data practices, this page will be updated and the date in the footer will reflect that change. to-you does not currently, and has no plans to, collect any user data.
